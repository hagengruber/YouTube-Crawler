Installation und Verwendung

Den YouTube-Crawler kann man auf zwei verschiedene Weisen verwenden:

  • Als zusätzliches Script im PHP-Projekt
  • Als Ajax-Schnittstelle

In PHP-Datei einbinden

Die Datei 'youtube-crawler.php' in die PHP-Script einbinden und die Funktion 'new_video' aufrufen.

Als Übergabeparameter wird der Link des YouTubers und die Anzahl an Videos übergeben.

Diese Methode belastet die Performance der Website enorm

<?php

include_once('./YouTube-Crawler/youtube-crawler.php');

$y = new ytcrawler;

$data = $y->new_video('https://www.youtube.com/user/MrSuicideSheep', 10);

for($i = 0; $i != count($data); $i++)

echo ' <a href="' . $data[$i]['link'] . '"> <img src="' . $data[$i]['thumbnail'] . '">' . $data[$i]['name'] . ' </a> <br> ';

?>

Als Ajax-Schnittstelle (Empfohlen)

Mit XMLHttpRequest 'yt_Request' (mit dem Inhalt des Links) und 'count' (Anzahl der Videos) als FormData() per POST an die 'youtube-crawler.php' Datei senden.

Die Daten werden als JSON-Array an die JavaScript-Datei mit dem Status '200 OK' zurückgesendet.

Da das Script asynchron mit dem Laden der Website läuft, wird die Performance dadurch nicht beeinträchtigt

<script>

var request = new XMLHttpRequest;

var data = new FormData();

data.append('yt_Request', 'https://www.youtube.com/user/MrSuicideSheep');

data.append('count', 5);

request.addEventListener('load', function(event) {

var jsonData = JSON.parse(request.responseText);

for (var i = 0; i < jsonData.length; i++) {

var data = jsonData[i];

console.log(data.link);

console.log(data.name);

console.log(data.thumbnail);

}

});

request.open('POST', './YouTube-Crawler/youtube-crawler.php');

request.send(data);

</script>
